# VIBYN E-commerce Website

A modern, premium e-commerce website for VIBYN, a UK-based streetwear and lifestyle clothing brand.

## Features

- Responsive design for all devices
- Modern, minimalistic aesthetic with streetwear vibes
- Product catalog with filtering and sorting
- Detailed product pages with image galleries
- Shopping cart and checkout flow
- Custom DTF (Direct to Film) design showcase
- About page with brand story
- Contact page with inquiry form
- Legal pages (Privacy Policy, Terms & Conditions)

## Tech Stack

- [Next.js 14](https://nextjs.org/) with App Router
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [React](https://reactjs.org/)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Project Structure

src/
├── app/              # Next.js app router pages

## Deployment

This site can be deployed to any platform that supports Next.js, such as Vercel, Netlify, or a custom server.

For Vercel deployment:
1. Push code to GitHub
2. Connect repository to Vercel
3. Set environment variables if needed
4. Deploy!

## Brand Guidelines

- **Color Palette**: Black, white, and metallic tones (silver/grey)
- **Typography**: Sans-serif, bold fonts - clean and elegant
- **Design**: Spacious, visual-driven, mobile-friendly
- **Tone**: Urban, confident, stylish, and youthful
